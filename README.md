## Unimelb Grade Statistics

A web application that helps students view grades that other students have attained in the past.

Author: Pulkit Aggarwal

Tech Stack

- React.js
- Node.js
- MongoDB

Deployed Application: https://grades.tech/

### Development

#### Frontend -

```
cd client
```

```
npm install
```

```
npm start
```

#### Backend -

```
cd api
```

```
npm install
```

```
nodemon index.js
```
